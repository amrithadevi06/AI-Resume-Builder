# AI Resume Builder

AI Resume Builder is a web application that helps users create professional resumes quickly by entering their personal, education, and skills information. The system generates a structured resume automatically.

## Live Application
Try the app here:
https://ai-resume-builder-jbbhzopjch4xt3ggymp3kx.streamlit.app

## GitHub Repository
https://github.com/amrithadevi06/AI-Resume-Builder

## Features
- Simple and interactive interface
- Generate professional resumes instantly
- Input personal details, education, and skills
- Download formatted resume

## Technologies Used
- Python
- Streamlit

## How to Run Locally

1. Clone the repository

git clone https://github.com/amrithadevi06/AI-Resume-Builder.git

2. Navigate to the project folder

cd AI-Resume-Builder

3. Install dependencies

pip install -r requirements.txt

4. Run the Streamlit app

streamlit run resume_builder_app.py
