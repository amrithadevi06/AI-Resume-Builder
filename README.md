# AI Resume Builder

AI Resume Builder is a web application that helps users generate professional resumes easily by entering their details.

## Features
- Generate resumes quickly
- Simple user interface
- Download resume as PDF
- Built using Python and Streamlit

## Technologies Used
- Python
- Streamlit
